# 页面照片放置说明

`p1` 只是原始素材库，不是网页入口。

你后续要手动把照片分到这 4 个网页对应的位置：

## 1. Landscape

放到这里：

```text
photos/landscape/
```

## 2. 大舞台 の一歩

放到这里：

```text
photos/da-butai/
```

## 3. Polaroid

放到这里：

```text
photos/polaroid/
```

## 4. About

放到这里：

```text
photos/about/
```

## 5. 规则

- 你自己决定哪些照片进哪个页面
- 页面里先不自动分类
- 先手动把照片复制到对应文件夹
- 文件名尽量保持简单

## 6. 当前状态

这些文件夹现在可以先为空。等你分好照片后，我再帮你把每个页面的图片位接上。
