# 页面照片放置说明

`p1` 只是原始素材库，不是网页入口。现在它放在站点根目录外面的 `../p1-source/`，不参与公开发布。

你后续手动把照片分到这几个网页对应的位置：

## 1. Home

放到这里：

```text
photos/hero/main.jpg
```

## 2. Landscape

放到这里：

```text
photos/landscape/
```

## 3. 大舞台 の一歩

放到这里：

```text
photos/da-butai/
```

## 4. Polaroid

放到这里：

```text
photos/polaroid/
```

## 5. About

放到这里：

```text
photos/about/
```

## 6. 规则

- 你自己决定哪些照片进哪个页面
- 页面里不做自动分类
- 先手动把照片复制到对应文件夹
- 再按 `photo-01.jpg`、`photo-02.jpg` 这种顺序改名
- About 页只放一张代表照片

## 7. 当前状态

这些文件夹现在可以先为空。等你分好照片后，再把每个页面的图片位接上。
