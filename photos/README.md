# 照片放置说明

你只需要把照片放到这个 `photos` 文件夹里，网站就会自动读取固定文件名。

## 1. 首页主照片

放到这里：

```
![[main.jpg]]
```

## 2. 系列一：Hong Kong Street 2025

放到这里：

```text
photos/hong-kong-street/cover.jpg
photos/hong-kong-street/photo-01.jpg
photos/hong-kong-street/photo-02.jpg
photos/hong-kong-street/photo-03.jpg
photos/hong-kong-street/photo-04.jpg
photos/hong-kong-street/photo-05.jpg
photos/hong-kong-street/photo-06.jpg
```

## 3. 系列二：City Night Fragments

放到这里：

```text
photos/city-night/cover.jpg
photos/city-night/photo-01.jpg
photos/city-night/photo-02.jpg
photos/city-night/photo-03.jpg
photos/city-night/photo-04.jpg
photos/city-night/photo-05.jpg
photos/city-night/photo-06.jpg
```

## 4. 照片建议

- 文件名先不要改，直接按上面的名字保存
- 第一版统一用 `.jpg`
- 单张照片建议控制在 1MB 到 3MB
- 长边建议 1600px 到 2000px

## 5. 你现在最简单的做法

1. 先选 1 张首页主照片
2. 再选 2 个系列
3. 每个系列先选 6 张
4. 按上面的文件名放进去

放进去之后，重新打开 `index.html` 就能看到真实照片。
