# 照片放置说明

你只需要把照片放到这个 `photos` 文件夹里，对应页面会读取固定文件名。

## 1. 首页主照片

放到这里：

```text
photos/hero/main.jpg
```

## 2. Landscape

放到这里：

```text
photos/landscape/photo-01.jpg
photos/landscape/photo-02.jpg
...
photos/landscape/photo-16.jpg
```

## 3. 大舞台 の一歩

放到这里：

```text
photos/da-butai/photo-01.jpg
photos/da-butai/photo-02.jpg
...
photos/da-butai/photo-10.jpg
```

## 4. Polaroid

放到这里：

```text
photos/polaroid/photo-01.jpg
photos/polaroid/photo-02.jpg
...
photos/polaroid/photo-15.jpg
```

## 5. About

放到这里：

```text
photos/about/photo-01.jpg
```

## 6. 规则

- 文件名保持简单
- 公开页面先只用 `.jpg` 和生成好的轻量版本
- `p1` 只是原始素材库，不是网页入口
